Ketentuan pembuatan RESTful API dan penggunaannya

### Standart RESTful API

| No | Verb | Path | Action |
|----|------|------|--------|
| 1. |  GET | /USER| INDEX  | 
| 2. |  GET | /USER/CREATE| CREATE  | 
| 3. |  POST | /USER| STORE  | 
| 4. |  GET | /USER/{USERID} | SHOW  | 
| 5. |  GET | /USER/{USERID}/EDIT| EDIT  | 
| 6. |  PUT | /USER/{USERID}| UPDATE  | 
| 7. |  DELETE | /USER/{USERID}| DELETE  |
